## 形容词

### 形容词作谓语

- 花はきれいです　花很美
    - きれいです作谓语

- きれいな花　美丽的花

### な形容词

- ハンサム　handsome 
    - ハンサム**な**男（おとこ）

- きれい　漂亮，干净
    - きれい**な**傘 
    - きれいな部屋
    - 对应汉字是**綺麗**

- 静（しず）か　安静
    - 静かな図書館

- にぎやか　热闹
    - にぎやかな学校

- 有名（ゆうめい）
    - ゆうめいな人
    - 中国のパンダは有名です　中国的大熊猫很有名
    - 日本のアニメは有名です　日本的动画很有名

- 親切（しんせつ）
    - 親切な人

- 元気　げんき
    - お元気ですか　你好吗
    - げんきです　
    - 元気つ子　活力四射的人

- 暇　ひま　空闲
    - 暇な日曜日　空闲的周日

- 便利　べんり　

- すてき　很赞的
    - すてきな夜景　极美的夜景
    - すてきな旅行　很棒的旅行

### い形容词

- 大きい（おおきい）　大的
    - 大きい鞄　大的包

- 小さい（ちいさい）　小的
    - 小さい声（こえ）　小声

- 新しい（あたらしい）　新的
    - 新しい携帯　新手机

- 古い（ふるい）　旧的
    - 弟に古いカメラをあげました　我把旧相机给了弟弟

- いい/よい　好的
    - いい子　好孩子
    - いい人　好人
    - いいアイデア　good idea 
    - いいですね　好啊

- 悪い（わるい）　不好的
    - 悪い人　坏人
    - 悪い夢　噩梦

- 暑い（あつい）/ 熱（あつ）い　热/烫
    - あつっ/あちっ 口语化的表达

- 寒い（さむい）/ 冷たい（つめたい）　冷的/凉的
    - 寒い冬　寒冷的冬天
    - 冷たい雨が　
    - 冷たい心　冰冷的心

- 難しい（むずかしい）　难的
    - 質問しつもん　问题
    - 難しい質問　难题

- 易しい（やさしい）　简单的
    - 日本語は難しいですか、易しいですか。　日语是难还是易？

- 高い（たかい）　高的，贵的
    - 高いビル　高楼大厦
    - 日本の果物（くだもの）は高いです。　日本的水果很贵

- 安い（やすい）/ 低い（ひくい）　便宜/低，矮
    - 中国の果物は安いです　中国的水果很便宜
    - 背が低い　个子矮

- 面白い（おもしろい）　好玩儿，有趣
    - 面白い映画　有趣的电影
    - あの人、面白いですね

- おいしい　好吃的
    - おいしいラーメン　好吃的拉面

- 忙しい（いそがしい）　忙
    - 毎日忙しいです　每天都很忙
    - 毎日暇ですか　每天都很闲吗

- 楽しい　たのしい　愉快的
    - 楽しい旅行　

- 白い　しろい
    - 白い雪（ゆき）　白色的雪

- 黒い　くろい
    - 黒いインク　black ink 

- 赤い　あかい
    - 赤いりんご　red apple 

- 青い　あおい　blue
    - 青い空（そら）　blue sky 

- い adj. ends with い
    - きれい　是个例外，是な形容词

### 单词

- さくら　sakura 
    - 日本のさくらは有名です　

- 山　やま　
    - 山田　やまだ

- 町　まち　城市
    - にぎやかな町　热闹的城市

- 食べ物　たべもの　食物🍜
    - おいしい食べ物　美食

- 車　くるま　🚗

- 所　ところ　地方，场所
    - いつもの所　老地方
    - いつも　总是，经常
    - いつもの所で会いましょう　老地方见

- 寮　りょう　宿舍
    - 大学の寮　大学宿舍

- 勉強　べんきょう　名词的学习
    - 勉強します　是动词
    - 日本語の勉強、楽しですね　日语学习真有趣

- 生活　せいかつ
    - 忙しい生活　忙碌的生活

- 仕事　しごと　工作
    - お仕事　询问对方的工作
    - 仕事をします　工作的动词

- どう　怎么样
    - このネクタイはどうですか　这条领带如何

- どんな　什么样的+名词
    - 福山さんはどんな人ですか　福山先生是个怎样的人

- どれ　哪个（从>=3个里选择）
    - どれかいいですか　选哪个好呢
    - 2选1，用どちら
    - ABC、どれが好きですか　
    - AB、どちらが好きですか　

- とても　very
    - 彼はとてもいい人です　he is a very good person 

- あまり　不太怎么，用于否定，可以读作あんまり

- そして　然后，用于连接句子，一般是并列关系
    - それから　更强调先后顺序
    - パン、そして、ミルクをください　请给我面包和牛奶

- XXが，xx but
    - この鞄は綺麗ですが、たかいです　this bag is very beautiful, but very expansive 
    - その人はハンサムですが、性格悪（わる）いです　那个人虽然长得帅，但是性格太差了

## 语法

- さくらは綺麗です　樱花很漂亮
    - きれい是な形容词
    - 野田（のだ）さんは親切です　野田小姐很亲切
    - お爺さんはとても元気です　爷爷身体很健康

```
大阪はにぎやかですか　大阪很热闹吗
はい、にぎやかです　是的，很热闹
```
```
ネットスクールは便利ですか　网校方便吗
はい、便利です　是的，很方便
```
```
木村拓哉はハンサムですか　　
はい、ハンサムです　
```
- 否定回答是いいえ、あまりハンサムじゃありませんえ
- な形容词+じゃありません表示否定
- `静かです`的否定是`静かじゃありません`
- `有名です`的否定是`有名じゃありません`

```
琵琶湖の水は、綺麗ですか　琵琶湖的水干净吗
いいえ、あまり綺麗じゃありません　不，不怎么干净
```
- あまり+否定式表示不太怎么样，语气缓和一些
- この映画（えいが）はあまり有名じゃありません　这部电影不是很有名
- その町は、あまりにぎやかじゃありません　那个城镇不是很热闹
- じゃありません也可以说成ではありません

- 富士山は高いです　富士山很高
    - 高い是い形容词

```
ペキンは今、寒いですか　北京现在冷吗
はい、とても寒いです 是的，非常冷

シャンハイも寒いですか　上海也冷吗
いいえ、あまり寒くないです　不，不太冷
```
- い形容词的否定形式，把`寒い`变成了`寒くない`
- `新しいです`->`新しくないです`
- `暑いです`->`暑くないです`
- `難しいです`->`難しくないです`
- `楽しいです`->`楽しくないです`
- 特殊否定形式`いいです`->`よくないです`

```
その辞書はいいですか　
いいえ、あまりよくないです　
```
- い形容词去掉い＋くないです🟰去掉い＋くありません
- よくないです＝よくありません

```
東京の地下鉄はどうですか　东京地铁怎么样
綺麗です。そして、便利です　很干净，而且很方便
```
- はどうですか询问地方/人物的感觉
```
日本の生活はどうですか　日本生活感觉如何
楽しいです　很开心
```
```
ネットスクールはどうですか　　网校感觉怎么样
とても便利です。そして、楽しいです　很便利，而且很有趣
```

- さくらは綺麗な花です　樱花是美丽的花
    - な形容词+な+名词

- 富士山は高い山です　fuji mount. is a high mount 
    - い adj + n. 

```
昨日（きのう）、映画を見ました　watched movie yesterday
どんな映画ですか　　which one
『七人の侍』です。古いですが、とても面白い映画です　七武士, it's old but interesting 
```
- どんな询问属性
- 古いです**が**, が跟在句子的结尾，表示转折
- 日本語の勉強は難しいです**が**、とても楽しいです　日语的学习虽然很难，也很愉快

```
奈良はどんな町ですか　奈良是一座什么样子的城市
古い町です　是个古老的城市
```

```
山田さんはどんな人ですか　山田是个什么样子的人？
面白い人です　有趣的人
```

```
ミラーさんの傘は、どれですか　米勒先生的伞是哪一把
あの青い傘です　那把蓝色的
```
- どれ是从>=3中选
- どちら是二选一
