## 动词

- 起（お）きます　起床
    - 寝（ね）ます　睡觉

- 働（はたら）きます　工作
    - 休（やす）みます　休息
        - おやすみなさい　晚安

- 勉強します　学习

- 手紙てがみ　信件

- 結束 けつそく　团结一致

- 大丈夫 だいじょうぶ　没关系

- 終（お）わります　结束
    - 終わりましょう　

- 动词的ます形比较有礼貌，常用

## 文型

- 今四時五分です　　
    - 现在是xxx点几分

```
ニューヨークは今何時ですか
午前10時10分です
```
- 纽约几点了
- 上午10点10分

- 0時　れいじ 也是 晚上12点

- 私は9時から5時まで働きます　　我的工作时间是从9点到5点
    - 9時から働（はたら）きます　从9点开始工作
    - 大坂からTokyoまで3時間かかります　　从大阪到东京需要3个小时
    -　銀行（ぎんこう）は9時から3時までです　　银行从9点开到3点
    - 昼休みは12時からです　　午休从12点开始
    - 今日の授業（じゅぎょう）はここまでです　今天的课就上到这里

```
銀行は何時から何時までですか
9時から3時までです

休みは何曜日ですか
土曜日と日曜日です
``` 
- 会社は何時から何時までですか　公司是几点到几点上班
- 図書館（としょかん）は何時までですか　　图书馆几点关门
- 朝7時から、夜10時までです
- 今日は何曜日ですか　　今天星期几

- 私は朝6時に起きます　　我早上六点起床
    - に表示动作发生的时间
    - 昨日、勉強（べんきょう）しました　昨天学习了，时间是昨天，没有数字，所以不加に

- 私は昨日勉強しました　
    - xxxました是ます的过去式
    - ます表示现在时和将来时，或者习惯性的事情
    - 今朝6時に起きました　　今天早上6点起床，用过去时

- ません　表示动作的否定
    - 起きません，不起床/不打算起床

- 过去否定 ませんでした
    - 起きませんでした　没有在过去某个时间起床
    - 今朝、6時におきませんでした　我没在6点起床

- 明日、休みます　　明天休息
    - 明天不用に
    - 将来时用ます

- 昨日、休みました　　昨天我请假了
    - ました表示过去

- 昨日、休みませんでした　昨天没休息

```
毎晩（まいばん）、何時に寝ますか
11時に寝ます
```
- に表示动作发生时间

```
昨日、勉強しましたか
はい、勉強しました/いいえ、勉強しませんでした
```

```
毎朝、何時に起きますか
6時に起きます
```
- 回答的时候需要重复动词

```
土曜日働きますか
いいえ、働きません
```

```
昨日、勉強しましたか
いいえ、勉強しませんでした
```

```
IMCの電話番号は何番ですか
123の456です
```
- の表示分割线

- ね加在句子末尾
    - 升调表示征求同意
    - 降调表示感叹

```
山田さんの電話番号は123の456です
123の456ですね
```
- ね表示确认

```
毎日10時ごろまで勉強します
大変（たいへん）ですね
```
- ね表示感叹
